# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-19

### Added
- Initial release with 17 widgets
- **Ticker** — Ticker Tape (scrolling) and Ticker (static grid)
- **Market Movers** — Crypto, Forex, and Stock market movers
- **Currency** — Currency Converter and Currency Cross Rates
- **Heatmap** — Crypto, Stock, and Currency heatmaps
- **Screener** — Crypto, Stock, Forex, Fund, CEX, DEX, and DR screeners
- Each widget demo includes 4 variations showcasing different settings, colors, and layouts
- Shared demo CSS and JS for consistent theme switching across all pages
- Full README with installation, customization reference, symbol format guide, and browser compatibility
